class Poirot.Poirot(pType='')¶

Poirot is an applciation that allow user to measure ultimate metrics like SNR and TTX against the ones obtained from simulations. In order to calculate the performance metric user need to set the UX metric and the Fields. The reference system of the UX and the field need t be oriented in the same way this means that x,y,z are defined as the nema standard (z is the B0 direction). Of course there will be trnsformation between the two spaces and can be set using the set transform
